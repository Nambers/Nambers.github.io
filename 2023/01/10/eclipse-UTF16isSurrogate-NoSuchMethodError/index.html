Log

版本

Eclipse IDE for Java Developers (includes Incubating components) Version: 2022-06 (4.24.0) Build id: 20220609-1112 openjdk 15

原因

测试了几遍, 发现是装了 Eclipse Web Developer Tools 导致的 Eclipse Web Developer Tools 3.26.0.v202204091445 org.eclipse.wst.web_ui.feature.feature.group

Solution

安装 Eclipse 2022-12 然后安装 2022-12 版本的 Eclipse Web Developer Tools
