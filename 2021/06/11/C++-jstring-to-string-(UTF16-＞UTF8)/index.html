C++ jstring to string (UTF16-＞UTF8)

2021-06-11 (Updated: 2024-02-09)
Note

JNI中jstring转码到std::string其实就是utf16转码到你选择的编码的过程，因为我C++程序要用utf8，所以这里我转成utf8
用到的转码库是utfcpp
这个库的使用方法就是下载source文件夹下全部内容，然后#include "utf8.h"在你的项目

实现思路可参照https://stackoverflow.com/questions/32205446/getting-true-utf-8-characters-in-java-jni
